Bold, Italic, Underline

Web Server merupakan sebuah aplikasi perangkat lunak. Aplikasi web server ini dijalankan pada sebuah komputer yang disebut dengan server Web Server adalah salh satu jenis perangkat lunak yang menyediakan layanan halaman (dokumen) web yang dapat diakses diseluruh dunia melalui internet

Analisis: ketika kita memasukkan tag b, maka tulsan akan menebal atau bold; ketika kita memasukkan tag i, maka tulisan akan miring atau italic; dan ketika kita memasukkan tag u, maka tulisan akan ada underline atau garis bawah
